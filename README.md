# AWS Serverless Lead Capture System

## 📌 Project Overview

This project demonstrates a fully serverless lead capture/contact form solution built on AWS.

The system accepts user submissions through API Gateway, processes requests using AWS Lambda, stores lead data in DynamoDB, and sends email notifications using Amazon SES.

The architecture is designed to be scalable, cost-efficient, and highly available without managing servers.

---

## 🏗️ AWS Services Used

- AWS Lambda
- Amazon API Gateway
- Amazon DynamoDB
- Amazon SES
- IAM
- CloudWatch Logs

---

## 🎯 Project Goals

- Build a fully serverless application on AWS
- Implement API-based backend processing
- Store structured lead data in DynamoDB
- Send automated email notifications
- Practice IAM permissions and monitoring
